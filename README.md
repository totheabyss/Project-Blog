## About the Project

This is a study project to practice consuming REST APIs using Node.js, Express, and Axios.
The focus is on CRUD operations, data validation, and HTTP routing best practices.

## Technologies Used

- Node.js
- Express
- Axios
- EJS

## API Used

This project consumes the public API **JSONPlaceholder**, which simulates CRUD operations.
The goal is to practice integration with REST APIs, not real data persistence.

## Features

- List items (GET)
- Create item (POST)
- Update item (PUT / PATCH)
- Delete item (DELETE)
- Validation of required fields

## How to Run the Project

```bash
npm install
npm start
```
